# README
 
本项目创建用于储存各类爬虫获取的数据，后在使用不断拓展添加各类数据生成接口(仅供学习，请勿用于其他途径)，本项目用于学习Demo，后续功能由学习人员实现（我仅指导需求），
主要功能包括并不限于:

## 爬虫

- 爬虫-世纪佳缘照片数据

- 爬虫-天眼查公司数据

- 爬虫-途虎车辆数据及保养相关数据

## 功能点

1.省市县数据
2.大小写数据转换
3.各类假数据生成
4.数据缓存，接口请求由原先2s降至2ms
5.异步接口
6.搜索/分页
6.简单统计
.....
